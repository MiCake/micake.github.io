# 工具集

MiCake 提供了一系列实用工具类，帮助您更高效地开发应用程序。本页面是工具集的导航页，点击各工具链接查看详细文档。

## 工具分类

### 缓存工具

| 工具 | 说明 |
|------|------|
| [BoundedLruCache](./BoundedLruCache.md) | 线程安全的 LRU 缓存，支持容量限制和分段策略 |

### 类型转换

| 工具 | 说明 |
|------|------|
| [ValueConverter](./ValueConverter.md) | 统一的类型转换接口，支持自定义转换器注册 |

### 查询工具

| 工具 | 说明 |
|------|------|
| [DynamicQuery](./DynamicQuery.md) | 动态查询构建器，支持基于属性特性自动生成过滤条件 |
| [Paging](./Paging.md) | 分页请求和响应模型，包含 IQueryable 扩展方法 |

### 弹性工具

| 工具 | 说明 |
|------|------|
| [CircuitBreaker](./CircuitBreaker.md) | 熔断器模式实现，保护系统免受外部服务故障影响 |

### 存储工具

| 工具 | 说明 |
|------|------|
| [DataDepositPool](./DataDepositPool.md) | 带容量限制的临时数据存储池 |

### 验证工具

| 工具 | 说明 |
|------|------|
| [CheckValue](./CheckValue.md) | 参数验证工具，提供常用的参数校验方法 |

### 随机工具

| 工具 | 说明 |
|------|------|
| [RandomHelper](./RandomHelper.md) | 线程安全的随机数生成和随机选择工具 |

### 扩展方法

| 工具 | 说明 |
|------|------|
| [Extensions](./Extensions.md) | 字符串、集合等常用扩展方法 |

## 快速开始

### 命名空间

```csharp
// 缓存
using MiCake.Util.Cache;

// 类型转换
using MiCake.Util.Convert;

// 动态查询和分页
using MiCake.Util.Query.Dynamic;
using MiCake.Util.Query.Paging;

// 弹性工具
using MiCake.Util.Resilience;

// 存储
using MiCake.Util.Store;

// 验证和随机
using MiCake.Util;

// 扩展方法
using MiCake.Util.Extensions;
```

### 常用示例

```csharp
// 参数验证
CheckValue.NotNullOrEmpty(name, nameof(name));

// 类型转换
int value = ValueConverter.Convert<string, int>("123");

// 字符串扩展
bool isEmpty = str.IsNullOrWhiteSpace();
string camel = "HelloWorld".ToCamelCase();

// 集合扩展
list.AddIfNotContains(item);

// 随机选择
var item = RandomHelper.GetRandomOfList(items);
```


## 最佳实践

1. **合理使用缓存**：缓存不经常变化的数据
2. **使用熔断器保护外部调用**：并提供降级方案
3. **工具类使用单例**：在 DI 容器中注册为单例
4. **参数验证前置**：在方法开头进行参数验证
5. **使用扩展方法**：简化常见操作
